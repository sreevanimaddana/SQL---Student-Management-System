# SQL Student Management System

## Project Overview
This project manages student, course, enrollment, and marks data using SQL.

## Tools Used
- SQL
- SQL Fiddle

## Database Tables
- Students
- Courses
- Enrollments
- Marks

## SQL Concepts Used
- SELECT
- JOIN
- WHERE
- GROUP BY
- ORDER BY
- Aggregate functions

## Project Outcome
Designed a relational database and extracted insights such as
top-performing students, course-wise averages, and enrollments.
